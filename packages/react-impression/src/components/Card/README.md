**外轮廓类型：Card 组件外轮廓可以设为外阴影、外边框和无边框三种类型。默认为外阴影**

```js
<Row>
  <Col>
    <Card>
      <CardBlock>
        <p>默认为外阴影类型</p>
        <Button theme="primary">好的</Button>
      </CardBlock>
    </Card>
  </Col>
  <Col>
    <Card outline="border">
      <CardBlock>
        <p>外边框类型</p>
        <Button theme="primary">好的</Button>
      </CardBlock>
    </Card>
  </Col>
  <Col>
    <Card outline="none">
      <CardBlock>
        <p>无边框类型，可以用来布局（不建议）</p>
        <Button theme="primary">好的</Button>
      </CardBlock>
    </Card>
  </Col>
</Row>
```
