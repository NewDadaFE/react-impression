### Example

**Basic**

```js
<Form type='inline'>
  <Form.Group>
    <label>defaultChecked:</label>
    <Checkbox ref='checkbox' defaultChecked>
      remember me
    </Checkbox>
  </Form.Group>
</Form>
```

**Disabled**

```js
<Form type='inline'>
  <Form.Group>
    <label>disabled:</label>
    <Checkbox disabled>remember me</Checkbox>
  </Form.Group>
</Form>
```
