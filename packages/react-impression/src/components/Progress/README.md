### 示例

**基本用法**

```js
<Row>
  <Col>
    <Progress value='1' />
  </Col>
</Row>
<Row>
  <Col>
    <Progress value='25' />
  </Col>
</Row>
<Row>
  <Col>
    <Progress value='50' />
  </Col>
</Row>
<Row>
  <Col>
    <Progress value='75' />
  </Col>
</Row>
```

**自定义主题**

```js
<Row>
  <Col>
    <Progress value='50' />
  </Col>
</Row>
<Row>
  <Col>
    <Progress value='50' theme='success' />
  </Col>
</Row>
<Row>
  <Col>
    <Progress value='50' theme='warning' />
  </Col>
</Row>
<Row>
  <Col>
    <Progress value='50' theme='danger' />
  </Col>
</Row>
```
