### Examples

**Basic**

```js
<Row>
  <Col>
    <Progress value='1' />
  </Col>
</Row>
<Row>
  <Col>
    <Progress value='25' />
  </Col>
</Row>
<Row>
  <Col>
    <Progress value='50' />
  </Col>
</Row>
<Row>
  <Col>
    <Progress value='75' />
  </Col>
</Row>
```
