### 示例

**自适应模式**

```js
<Image fluid src="https://placehold.it/2500x350" />
```

**多形状模式**

```js
<Row>
  <Col>
    <Image rounded src="https://placehold.it/200x200" />
  </Col>
  <Col>
    <Image circle src="https://placehold.it/200x200" />
  </Col>
  <Col>
    <Image thumbnail src="https://placehold.it/200x200" />
  </Col>
</Row>
```
