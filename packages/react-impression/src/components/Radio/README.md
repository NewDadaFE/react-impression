### Examples

**Basic**

```js
<Form>
  <Form.Group>
    <label>Radio</label>
    <Radio ref="radio" value={1} onChange={this.onChangeHandle}>
      single
    </Radio>
  </Form.Group>
</Form>
```

**Disabled**

```js
<Form>
  <Form.Group>
    <label>Disabled</label>
    <Radio disabled>Yes</Radio>
  </Form.Group>
</Form>
```
