### 代码示例

**横向默认布局**

```js
<Flex direction="row">
  <FlexItem flex={2}>
    <div>new DADA</div>
  </FlexItem>
  <FlexItem>
    <div>new DADA</div>
  </FlexItem>
  <FlexItem>
    <div>new DADA</div>
  </FlexItem>
</Flex>
```

<Flex direction="column" align="top">
  <FlexItem>
    <div>new DADA</div>
  </FlexItem>
  <FlexItem>
    <div>new DADA</div>
  </FlexItem>
  <FlexItem>
    <div>new DADA</div>
  </FlexItem>
</Flex>
